# BOJ Search by problem-number

## v1.0.0
MacOS App인 Alfred4에서 동작합니다.  
문제번호만을 이용하여 문제에 대한 정보를 추출합니다.  
추출한 정보는 {'title':['문제번호','문제명'], 'subtitle':['제출수','정답률']}입니다.
